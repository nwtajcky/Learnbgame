Clear Stashes

shortcut QWERTY xc, QWERTZ yc, only in object mode

The Clear Stash tool is used to remove a stash from an object.
This does not delete the stash completely, it only removes the association with the current object.

If the cleared stash is not associated with any other object, it becomes an Orphan stash.

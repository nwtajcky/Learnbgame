Requirements

  • Blender 2.79b

Note

Earlier versions as well as experimental builds may also work, but are untested.

Only stable releases are supported, the latest of which is 2.79b.
Use the addon at your own risk with the experimental builds.

Latest

The latest version of MESHmachine is 0.6 - available on Gumroad and Blender Market.
See this page the learn what's new in the latest versions, or see the changelog for the full release history.

Instructions

Install from File

  • start Blender, pressCTRL + ALT + U to open the user preferences
  • switch to the Add-ons tab and click the Install Add-on from file... button at the bottom
  • locate the downloaded MESHmachine_v0.6.zip file and double-click it
  • activate the addon by ticking the checkbox and hit the Save User Settings button at the bottom
  • press X to bring up the MESHmachine menu in either oject or edit mode

Manual Installation (alternative)

  • extract the MESHmachine_v0.6.zip archive
  • move the extracted MESHmachine folder and its contents to your addons folder in the user scripts location
  • start Blender, find the addon in preferences in the Add-ons tab and activate it, save your user config
  • press X to bring up the MESHmachine menu in either oject or edit mode

Addons Folder (in user scripts)

Linux: /home/name/.config/blender/2.79/scripts/addons
MacOS: /Users/name/Library/Application Support/Blender/2.79/scripts/addons1
Windows: C:\Users\name\AppData\Roaming\Blender Foundation\Blender\2.78\scripts\addons

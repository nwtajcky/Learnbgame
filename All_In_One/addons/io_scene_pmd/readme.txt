
Blender2.57でpmdファイルを読み込むスクリプトです。
Windows XPで動作確認済みです。

・設置方法
「Blender2.57のインストールフォルダ」→「2.57」→[scripts」→「addons」にフォルダごと設置して、Blenderを再起動してください。


・注意
Blender2.57でのみ対応しています。下位バージョンでは動作しません。
通常版は形状とマテリアルのみ、Beta版はボーン読み込みに対応してます。
vmdファイルは読み込めません。

Blenderの仕様でマテリアルは一つのオブジェクトに16個までしか割り当てることが出来ません。
マテリアル数が16個を超えた場合は16個毎に新たなオブジェクトが作成されます。

詳しくはhttp://pr0jectze10.tuzigiri.com/simpleVC_20101031222442.htmlを参照してください。

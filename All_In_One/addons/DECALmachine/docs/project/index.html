Project

shortcut dd1

Project conforms a decal to the curved surface of a non-decal target object.
Projected decals will keep their original, unprojected Decal as a backup.

Keep in mind

Decals on flat surfaces DO NOT need to be projected.
And neither do decals, covering only a single polygon on the target object.

Selection

Any number of Decal objects.
Already projected and sliced Decals among the selection will be ignored.

Modes

Default: Project Selected Decals on Surface
ALT: Manually adjust projection depth
CTRL: Use UV Project instead of UV Transfer
SHIFT: Shrinkwrap

Using Project

  1. In the case of dd, you need to also click with the mouse afterwards, otherwise the pie menu will stay open, as it is triggered with the second d as well, unfortunately. ↩

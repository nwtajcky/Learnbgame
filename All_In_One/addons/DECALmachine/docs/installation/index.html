Requirements

  • Blender 2.80, f606f588455e (May 20th)

Warning

Blender 2.80 is in BETA right now. No assurences can be made, that DECALmachine keeps working with every new daily build.
Use the recommended build above, and update on your own risk.

MacOS

MacOS users should install Blender properly, by following the official instructions.
Avoid running it just from the Downloads folder!
Note that, for dragging of files and folders, you need to hold down the command key.
This will ensure AppTranslocation1 is avoided.

Latest

The latest version of DECALmachine is 1.8.5 - available on Gumroad and Blender Market.
See this page to learn what's new in the latest versions, or see the changelog for the full release history.

Instructions

Install from File

  • start Blender, bring up preferences
  • switch to the Add-ons tab and click the Install... button at the top right
  • locate the downloaded DECALmachine_1.8.5.zip file and double-click it
  • activate the addon by ticking the checkbox and hit the Save User Settings button at the bottom
  • optionally, switch to the Decal Creation tab, and install PIL
  • close prefrences and in the 3D View press the D key to bring up the DECALmachine pie menu in object mode

Manual Installation (alternative)

  • extract the DECALmachine_1.8.5.zip file
  • move the extracted DECALmachine folder and its contents to your addons folder in the user scripts location
  • start Blender, find the addon in preferences in the Add-ons tab and activate it, save your user config
  • optionally, switch to the Decal Creation tab, and install PIL
  • close prefrences and in the 3D View press the D key to bring up the DECALmachine pie menu in object mode

Addons Folder (in user scripts)

Linux: /home/name/.config/blender/2.80/scripts/addons
MacOS: /Users/name/Library/Application Support/Blender/2.80/scripts/addons
Windows: C:\Users\name\AppData\Roaming\Blender Foundation\Blender\2.80\scripts\addons

  1. Learn more about AppTranslocation. ↩

clipLine
========


Functionality
-------------
Clips the line against the image rectangle.


Inputs
------
- imgRect_in – Image rectangle.
- pt1_in – First point of the line segment.
- pt2_in – Second point of the line segment.


Outputs
-------
- pt1_out – Pt1 output.
- pt2_out – Pt2 output.
- retval_out – Return value.

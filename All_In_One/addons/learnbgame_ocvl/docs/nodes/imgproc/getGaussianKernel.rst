getGaussianKernel
=================


Functionality
-------------
Returns Gaussian filter coefficients.


Inputs
------
- ksize_in – Aperture size. It should be odd.
- ktype_in – Type of filter coefficients. It can be CV_32f or CV_64F.
- sigma_in – Gaussian standard deviation.


Outputs
-------
- kernel_out – Output kernel.
